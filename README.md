# Mushroom Research Website

This is the official research website for **A Predictive Analytics Framework for Optimizing Mushroom Cultivation**.

**Project ID:** 25-26J-211  
**Institution:** Sri Lanka Institute of Information Technology  
**Technology:** Vite + React

## Project Overview

This project supports mushroom farmers through an integrated smart cultivation framework. The system includes:

- Environmental monitoring and forecasting
- Mushroom variety recommendation
- Mushroom type classification
- Growth-stage prediction
- Disease detection
- Treatment recommendation
- Severity monitoring
